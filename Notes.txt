Select a folder that has a git repo in it to push




…or create a new repository on the command line
echo "# test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:Ahmed-Fouad0440/test.git
git push -u origin main


…or push an existing repository from the command line
git remote add origin git@github.com:Ahmed-Fouad0440/test.git
git branch -M main
git push -u origin main
